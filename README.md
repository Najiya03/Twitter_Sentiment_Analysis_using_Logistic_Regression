# Twitter_Sentiment_Analysis_using_Logistic_Regression

##  Project Overview
This project performs **Sentiment Analysis on Twitter data** using **Natural Language Processing (NLP)** and **Machine Learning (ML)** techniques.  
The aim is to classify tweets into sentiment categories such as **Positive** or **Negative** based on the text content.

This repository demonstrates a full ML workflow including:
- Data preprocessing and cleaning
- Feature extraction
- Model training
- Evaluation and analysis
